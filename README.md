# My-Buddy is a Chatbot 🤖

A modern, responsive, and intelligent Chatbot web application powered by the **Google Gemini 2.5 Flash API**. 

This mini-project provides an elegant chat popup interface allowing users to converse with an AI assistant. It supports text input, interactive emojis, and image attachments.

## 🚀 Features
- **Responsive Popup Interface:** A beautiful floating chat widget that expands on click.
- **Gemini AI Integration:** Generates fast, context-aware responses using the Gemini 2.5 Flash API.
- **Text & Image Support:** Attach image files to prompt the AI visually.
- **Emoji Support:** Integrated with `emoji-mart` for easy emoji selection.
- **Chat History:** Maintains context of the conversation during the session.
- **Modern UI/UX:** Typing indicators, smooth scrolling, clean gradient backgrounds, and intuitive file upload previews.

## 🛠️ Technologies Used
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox layout, CSS variables, interactive hover states, beautiful gradients, and media queries for mobile responsiveness.
- **Vanilla JavaScript:** 
  - `fetch` API for making requests to the Gemini endpoint.
  - DOM manipulation and event handling.
  - Document attachments using `FileReader` and Base64 conversion.
- [Font Awesome](https://fontawesome.com/) (Icons)
- [Emoji Mart](https://github.com/missive/emoji-mart) (Emoji Picker)

## 💻 Getting Started
1. Clone or download the repository.
2. Ensure you have an active open internet connection (for loading CDN dependencies and making API requests).
3. Obtain a Google Gemini API Key from Google AI Studio.
4. Open the `script.js` file and replace the `API_KEY` with your actual Gemini API key.
5. Open `index.html` in your favorite web browser to start chatting!
